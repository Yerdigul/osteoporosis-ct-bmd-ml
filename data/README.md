# Dataset Access

The raw CT spine dataset is not stored in this GitHub repository because of file size and data-sharing restrictions.

Dataset access:
[[Google Drive Dataset Link Here]([https://drive.google.com/drive/folders/1nvgKHKbsb1RPx_q9nNGegVCpVOfvTI_T?usp=drive_link](https://drive.google.com/drive/folders/1raKxt0tVxq5LEmEJoBby-0Hk25G8MEoQ?usp=sharing))]

Folder structure:
- raw_data/: original CT scan / segmentation files
- processed_data/: cleaned outputs
- outputs/: BMD, T-score, and diagnosis results
- figures/: plots and visualizations
